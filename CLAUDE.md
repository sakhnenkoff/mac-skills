# mac-skills

Personal collection of reusable Mac setup skills for Claude Code.

## Available Skills

| Skill | Purpose |
|-------|---------|
| **github-setup** | Complete GitHub setup on a new Mac — SSH keys, multi-account SSH config, GitHub CLI installation and authentication |
| **github-email-filters** | Set up Gmail filters to silence noisy GitHub notification emails while keeping important ones (review requests, author replies, @mentions) |

## Adding New Skills

1. Create a directory under `.claude/skills/<skill-name>/`
2. Add a `SKILL.md` with frontmatter (`name`, `description`) and a procedure
3. Update this table
